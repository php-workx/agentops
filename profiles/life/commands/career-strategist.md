# /career-strategist - Career Planning

**Purpose:** Guide career strategy, competitive positioning, and application preparation.

**When to use:** Quarterly reviews, major applications (NVIDIA), career transitions, strategic planning.

---

## What This Command Does

Loads the `career-strategist` agent to help you:

1. **Update Career Strategy**
   - Review 10-year vision
   - Assess current positioning
   - Identify gaps and opportunities
   - Plan next career moves

2. **Prepare Applications**
   - NVIDIA positioning and narrative
   - Resume updates with latest achievements
   - Cover letter strategy
   - Interview preparation

3. **Competitive Analysis**
   - Understand market positioning
   - Identify unique value proposition
   - Benchmark against peers
   - Strengthen differentiators

4. **Strategic Planning**
   - Quarterly goal alignment
   - Skill development priorities
   - Visibility campaign planning
   - Network building strategy

---

- `life/operations/career-strategy.md` - Master career strategy document

- `life/operations/construct-cycle.md` - Quarterly rhythm and goals

## Typical Session Flow

### Phase 1: Context Loading (5-10 min)
Agent reads current career strategy and MCI to understand:
- Where you are now
- Where you want to go
- What you've accomplished recently
- What gaps exist

### Phase 2: Strategic Analysis (10-20 min)
Agent analyzes:
- Market positioning
- Competitive landscape
- Growth opportunities
- Strategic priorities

### Phase 3: Action Planning (10-20 min)
Agent creates:
- Updated career strategy document
- Specific action items with timeline
- Application preparation tasks (if relevant)
- Quarterly goals alignment

### Phase 4: Documentation (5-10 min)
Agent updates:
- `life/operations/career-strategy.md` with insights
- Session log with decisions and rationale
- Next steps checklist

**Total time:** 30-60 minutes

# General career review
