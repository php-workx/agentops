## üìñ Key Concepts

### Agents
Specialized AI components that perform specific tasks:
- **change-executor** - Implements code changes
- **spec-architect** - Creates technical specifications
- **continuous-validator** - Validates implementations
- [Full agent reference ‚Üí](./reference/agents/)

### Commands
User-facing commands for orchestration:
- `/plan` - Create execution plans
- `/implement` - Execute implementations
- `/validate` - Validate results
- [Full command reference ‚Üí](./reference/commands/)

### Workflows
Multi-agent orchestration patterns:
- **complete-cycle** - Full development cycle
- **quick-fix** - Fast bug fixes
- **debug-cycle** - Debugging workflow
- [Full workflow reference ‚Üí](../core/workflows/)

### Profiles
Domain-specific configurations:
- **base** - Core functionality
- **devops** - DevOps practices
- **product-dev** - Product development
- [Create your own ‚Üí](./CREATE_PROFILE.md)
